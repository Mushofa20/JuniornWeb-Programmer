WHAT DO WE LEARN?

Untuk membuat suatu website, yang perlu kita lakukan adalah:

buat struktur dasarnya terlebih dahulu,
yaitu dengan menggunakan HTML
(Hypertext Markup Language).
Kemudian, untuk membuat website yang lebih interaktif,
kita bisa menggunakan JavaScript dan CSS.

Kalau kita ingin membuat website yang lebih kompleks, maka:

kita dapat menggunakan PHP,
yaitu singkatan dari Hypertext Preprocessor.
Bahasa skrip PHP dapat dijalankan di web server dan
dikombinasikan dengan HTML, CSS, dan JavaScript.
Sedangkan database digunakan bila kita memiliki banyak data,
karena memudahkan dalam pengorganisasian dan pengelolaan data.
